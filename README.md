# Блоговое приложение на Django

## Описание
Вы создаете блоговое приложение для ведения списка статей. Приложение включает две модели: "Статья" и "Автор".

## Модель "Автор"
- Имя (CharField)
- Фамилия (CharField)
- Электронная почта (EmailField)

## Модель "Статья"
- Заголовок (CharField)
- Содержание (TextField)
- Дата публикации (DateTimeField, автоматически заполняемое текущей датой и временем)
- Автор (ForeignKey, связанный с моделью "Автор")

## Задачи
1. Создайте Django-приложение с указанными выше моделями.
2. Создайте административный интерфейс Django для управления моделями "Автор" и "Статья".
3. Создайте URL-маршруты для отображения списка всех статей и детальной страницы статьи. На детальной странице статьи должен отображаться заголовок, содержание и имя автора.
4. Создайте представления (views) для отображения списка статей и детальной страницы статьи. Используйте шаблоны для отображения данных.
5. Выведите список статей на главной странице.
6. Создайте форму для добавления новых статей в блог. Форма должна включать поля "Заголовок", "Содержание" и "Автор" (выпадающий список авторов).
7. Создайте URL-маршрут и представление для добавления новых статей через форму.
8. Добавьте базовую структуру HTML-шаблонов для отображения списка статей, детальной страницы статьи и формы добавления статьи.
